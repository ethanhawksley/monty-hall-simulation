# Monty Hall Simulation

## About

A high-performance simulation of the Monty Hall probability paradox written in Rust. Demonstrates how switching leads to winning the scenario twice as often as staying.

## Usage

1. Clone the repository:
   ```shell
   git clone https://github.com/ethanhawksley/monty-hall-simulation
   cd monty-hall-simulation/monty-hall-simulation
   ```
2. Compile and run:
   ```shell
   cargo run
   ```
